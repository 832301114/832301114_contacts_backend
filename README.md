# 通讯录系统后端

这是一个基于Spring Boot的通讯录系统后端API。

## 功能特性

- 联系人的增删改查
- 联系人搜索功能
- 数据验证
- RESTful API设计
- 内存数据库(H2)

## 技术栈

- Spring Boot 2.7.0
- Spring Data JPA
- H2 Database
- Maven

## 快速开始

### 环境要求

- Java 11+
- Maven 3.6+

### 运行步骤

1. 克隆项目
2. 进入项目目录
3. 运行命令：`mvn spring-boot:run`
4. 访问：http://localhost:8081
